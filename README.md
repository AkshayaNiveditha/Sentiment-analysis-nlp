# Sentiment Analysis using NLP

## 📌 Problem Statement
This project classifies text into positive or negative sentiment.

## ⚙️ Approach
- Used TF-IDF vectorization to convert text into numerical features
- Trained a Logistic Regression model
- Tested the model on new input text

## 🧠 Technologies Used
Python, Scikit-learn

## 📊 Example Output
Input: "I hate this"  
Output: Negative

## 🚀 Future Improvements
- Use Transformer models like BERT
- Build real-time chatbot using LLMs
- Improve accuracy with larger dataset
